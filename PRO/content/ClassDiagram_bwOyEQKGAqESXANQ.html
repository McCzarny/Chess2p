Elementy modelu
Nazwa Dokumentacja
  CompPosition : Klasa

Komponent pozycji. Pozycja jest określona w 3 wymiarach. 2 wymiary oznaczają pozycję na ekranie, a 3. głębokość.

  Component : Klasa

Abstrakcyjna klasa komponentu. Z niej dziedziczą poszczególne klasy używane przez moduły.

  GameObject : Klasa

Obiekt w grze. Składa się z komponentów, które są używane przez różne moduły.

  Engine : Klasa

Silnik gry. Zarządza działaniem aplikacji.

  IO : Klasa

Klasa odpowiedzialna za sterowanie.

  Renderer : Klasa

Moduł odpowiedziany za renderowanie obiektów gry.

  Game : Klasa

Główny obiekt aplikacji. Inicjalizuje silnik i grę.

  CompSprite : Klasa

Tekstura obiektu. Rysowana podczas wywołania onDraw.

  Scene : Klasa

Scena zawierająca moduły i obiekty sceny.

  Module : Klasa

Abstarakcyjna klasa modułu. Z niej dziedziczą poszczególne części silnika.

  Vector2F : Klasa

2-wymiarowy wektor float.

  Vector3F : Klasa

3-wymiarowy wektor float.

  Sprite : Klasa

Klasa tekstury.

  CompGUI : Klasa

Komponent GUI. Zawiera teksturę, która będzie wyrysowana na ekranie.

  Settings : Klasa

Ustawienia silnika.
